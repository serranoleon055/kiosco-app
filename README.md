# 🏪 Kiosco App

Sistema web de gestión de kiosco construido con React + TypeScript.

## 🚀 Demo

[kiosco-app-alpha.vercel.app](https://kiosco-app-alpha.vercel.app)

## ✨ Features

- Catálogo de productos con stock en tiempo real
- Carrito de compras con control de stock
- Eliminar productos individuales del carrito
- Filtro por categorías dinámico
- Persistencia con localStorage

## 🛠 Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite

## 📦 Instalación

```bash
git clone https://github.com/TU-USUARIO/kiosco-app.git
cd kiosco-app
npm install
npm run dev
```
